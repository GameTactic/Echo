# GameTactic Echo

## Settings up locally.

1. Git clone.
2. Use OpenJDK 11.
3. Run with `./mvnw spring-boot:run`


## Packing

1. `./mvnw package`
2. `./mvnw dockerfile:build`*
3. `./mvnw verify`
4. `./mvnw dockerfile:push`*
5. `./mvnw deploy`

\* _Only maintainers when doing release!_
